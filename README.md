# Shopify-demo_CORE-Plus

健康食品のECサイトを想定して制作した、Shopifyデモストア用のテーマ開発リポジトリです。  
Dawnをベースに、セクション追加・商品ページ改修・UI調整を行っています。

## 概要
Shopifyテーマのカスタマイズや改修対応を想定して制作したデモテーマです。  
単にデザインを形にするだけでなく、商品導線・回遊・見やすさを意識しながら実装しています。

## 主な実装内容
- ヒーローセクションのカスタマイズ
- 商品ページのカスタマイズ
- セクション追加
- 管理画面から調整しやすい block / setting の実装
- CSS / JavaScript によるUI調整

## 使用言語
- Shopify Liquid
- HTML
- CSS
- JavaScript

## 補足
デモ用リポジトリのため、一部画像・文言は検証用です。
